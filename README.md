# 🎧 Bossman Entertainment Booking Portal
### *Vico Green & McLight Official Digital Hub*

Built with passion for the movement. This portal is designed to streamline bookings, showcase live experiences, and manage the tour schedule for South Africa's finest Afro-Tech and Gqom duo.

---

## 🚀 Key Features
- **3D Glassmorphism Design:** A high-end, dark-mode aesthetic.
- **Live Gig Guide:** Easily editable schedule for management (Erick Khomo).
- **YouTube Integration:** Native embeds for high-quality live set streaming.
- **Dynamic Poster Grid:** Dedicated slots for event flyers and promotional content.
- **Mobile Optimized:** Designed for promoters to book straight from their phones.

---

## 📂 Folder Structure for Assets
To keep the site looking "dope," upload your files to `/public/assets/` using these names:

| Asset Type | Recommended Filename | Description |
| :--- | :--- | :--- |
| **Gigs** | `flyer1.jpg`, `flyer2.jpg` | Event posters |
| **Artists** | `vico.jpg`, `mclight.jpg` | Clean artist headshots |
| **Gallery** | `gallery1.jpg` to `gallery6.jpg` | Performance & lifestyle shots |

---

## 🛠️ Management Instructions (For Bossman)
To update the Gigs:
1. Open `App.js`.
2. Locate the `gigs` state array at the top.
3. Update the `date`, `name`, and `venue`.
4. The site will update instantly across all devices.
